# 二分查找

## 题目1

- <font color=orange>有序数组</font>A中，查找target值
- 查找到则返回索引，查找不到则返回-1

```java
/*基础版本实现*/
public int solution_01(int[] arr, int target) {
    int i = 0;
    int j = arr.length - 1;

    while (i <= j) {   // <=
        int m = (i + j) / 2;
        if (target < arr[m]) {
            j = m - 1;
        } else if (arr[m] < target) {
            i = m + 1;
        } else {
            return m;
        }
    }
    return -1;
}
```
